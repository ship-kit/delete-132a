# Clerk

Complete user management platform with pre-built UI components.

```bash
CLERK_SECRET_KEY=sk_test_your_secret_key
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_your_publishable_key
CLERK_WEBHOOK_SECRET=whsec_your_webhook_secret
```

Note:

- You can set `CLERK_PUBLISHABLE_KEY` without the `NEXT_PUBLIC_` prefix and Shipkit will automatically generate `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY` at build time for use on the client.
- The non-prefixed variable is not accessible on the client; only the generated `NEXT_PUBLIC_*` value is exposed.
